# Global Football Scouting Database & Recommendation Engine

An end-to-end data pipeline built to fetch, clean, and analyze historical seasonal data (past 3 seasons) across 19 global football tiers. It provides tactical recommendations using vector proximity modeling.

## Technical Stack
* **Language:** Python 3.11+
* **Database:** SQLite
* **Environment & Tools:** VS Code, Git, Python-dotenv
* **Mathematical & Data Stack:** Pandas, Scikit-learn (StandardScaler, Cosine Similarity)

## Targeted Competitions
* **South America:** Brazil (Série A, B, C), Argentina, Colombia, Chile, Uruguay, Ecuador, Peru, Bolivia, Paraguay, Venezuela.
* **North America:** MLS, Liga MX.
* **Europe:** English Premier League, EFL Championship, La Liga, Serie A, Ligue 1, Bundesliga, Eredivisie, Liga Portugal, Belgian Pro League, Allsvenskan, Russian Premier League, Ukrainian Premier League.
* **Asia:** J1 League, Saudi Pro League.
